# 부동산 세금 계산기 (tax-calc)

부동산 거래 시 필요한 세금을 계산하는 무료 웹 서비스. SEO + AdSense 수익 모델.

## 기술 스택
- Next.js 16 (App Router)
- TypeScript, Tailwind CSS v4
- Google AdSense + 카카오 애드핏 (수익화)

## 계산기 4종
- `/transfer-tax` — 양도소득세
- `/acquisition-tax` — 취득세
- `/property-tax` — 종합부동산세
- `/rental-income-tax` — 임대소득세

## 규칙
- 한국어 서비스 (lang="ko")
- "참고용" 면책 문구 필수
- 세율은 2026년 기준
